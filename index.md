## 💡 Frequent Questions

### Does this change my system power settings forever?
No. The changes apply only while the program runs. Once you close the program or restart your computer, your original sleep settings return.

### Can I run multiple instances?
Running the program once is sufficient. If you start the program multiple times, it may create conflicts with your system power API. Stick to one active window for the best results.

### Does it use much battery?
The software uses almost zero processor power. It performs a simple check-in with your operating system every few minutes. You will not notice a drain on your battery while using it.

### What if I see a warning when I run the file?
Windows identifies files downloaded from the internet as potential risks. Because this is a direct tool for advanced power management, Windows may show a standard safety prompt. Click "More info" and then "Run anyway" to start the application.
