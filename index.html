Projects

확인 가능한 영역

Instagram-like profile shell

브랜드, 프로필 아바타, 통계, 하이라이트를 중심으로 한 개인 페이지 구조를 복원했습니다.

03

Playable snake game

키보드로 조작 가능한 지렁이 게임과 점수, 최고 점수, 일시정지를 넣었습니다.

Keyboard-driven tetris

테트리스는 화살표, Z/X, Space, P, R 키로 조작하도록 붙였습니다.

Games

지렁이 게임 옆에 테트리스 게임을 붙였습니다

두 게임 모두 키보드 입력으로 조작할 수 있으며, 마지막으로 선택한 게임이 키를 받습니다.

Snake

지렁이 게임

방향키와 WASD로 움직입니다. 마지막으로 선택한 게임이 키보드 입력을 받습니다.

상태 대기 중
점수 0
최고 점수 0
폭발까지 05.0s
지렁이 게임을 선택하면 방향키로 시작합니다. 게임 오버, 일시정지, 재시작을 지원합니다.

Tetris

테트리스 게임

화살표, Z/X, Space, P, R로 조작합니다. 키보드 입력으로 블록을 돌리고 떨어뜨립니다.

상태 대기 중
점수 0
줄 0
레벨 1
테트리스 게임을 선택하면 키보드로 시작합니다. 좌우 이동, 회전, 하드 드롭, 일시정지, 재시작을 지원합니다.
